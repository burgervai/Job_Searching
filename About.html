About RiseBoard

A student-built job board for graduates and job seekers.

What is RiseBoard?

RiseBoard is a career platform built for students and graduates. It pulls live job listings from the Rise API so you can search and filter roles without signing up or dealing with ads.

This project was created for ISYS3004 Business Web and Mobile Technologies at Curtin University to demonstrate skills in HTML, CSS, JavaScript, and REST API integration.

Who is it for?

  • 🎓 Students

    Looking for internships or part-time roles while studying.

  • 🚀 Graduates

    Ready for full-time work after finishing a degree.

  • 🌍 Remote Workers

    Want to work from anywhere in the world.
